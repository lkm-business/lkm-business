# LKM_BUSINESS 🛒
### "Toi-même faut voir"

Site e-commerce complet — Produits physiques & Abonnements digitaux

## Technologies
- React 18 (Frontend)
- Node.js + Express (Backend API)
- PostgreSQL (Base de données)
- JWT (Authentification)
- Nodemailer (Emails)
- Stripe / Orange Money / Wave (Paiements)

## Démarrage rapide

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Remplis les variables dans .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
echo "REACT_APP_API_URL=http://localhost:5000/api" > .env.local
npm start
```
